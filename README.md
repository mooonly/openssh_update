                       openssh升级时，需先要开启telnet，确保telnet可以正常登陆。
                  这样当openssh升级出现问题的时候，还可以通过telnet登录到服务器操作。
                        此脚本需要您提供root账户的密码来自动进行telnet连接测试
                         可通过执行./run.sh host   password 的方式直接带入密码
                                    eg:./run.sh 192.168.1.1  123456
                                 或者直接执行./run.sh 来进行交互式获取
